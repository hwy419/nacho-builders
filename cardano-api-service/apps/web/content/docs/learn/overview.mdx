## Path Details

### Wallet Developer Path

Perfect for building:
- Mobile or web wallets
- Balance tracking dashboards
- Payment applications

**You'll learn:**
1. How to query UTxOs and calculate balances
2. Building and signing transactions
3. Tracking transaction status
4. Handling multiple addresses

**Prerequisites:** Basic programming knowledge in TypeScript or Python

---

### dApp Developer Path

Perfect for building:
- NFT marketplaces
- DeFi applications
- Gaming platforms

**You'll learn:**
1. Integrating with browser wallets (Nami, Eternl, etc.)
2. Building transactions with scripts
3. Evaluating transaction costs
4. Handling smart contract errors

**Prerequisites:** Wallet Developer path or equivalent knowledge

---

### Exchange Integrator Path

Perfect for building:
- Cryptocurrency exchanges
- Payment gateways
- Custodial services

**You'll learn:**
1. Real-time deposit monitoring with chain sync
2. Handling blockchain rollbacks
3. Secure withdrawal processing
4. High-availability patterns

**Prerequisites:** dApp Developer path or production backend experience

## Not Sure Which Path?

Answer these questions:

1. **Are you building a consumer application where users manage their own keys?**
   → Start with [Wallet Developer](/docs/learn/wallet-developer/overview)

2. **Do you need to interact with Plutus smart contracts?**
   → Start with [dApp Developer](/docs/learn/dapp-developer/overview)

3. **Do you need to process deposits/withdrawals for many users reliably?**
   → Start with [Exchange Integrator](/docs/learn/exchange-integrator/overview)

<Callout type="tip" title="Start Simple">
If you're completely new to Cardano, start with the Wallet Developer path. The concepts you learn there are foundational for the other paths.
</Callout>
